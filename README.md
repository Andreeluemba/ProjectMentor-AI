## 📌 O que é o ProjectMentor AI?

O **ProjectMentor AI** é uma aplicação web que actua como um mentor inteligente para programadores. Ajuda o utilizador a:

- ✅ Escolher e definir o tema do projeto
- ✅ Planear as etapas de desenvolvimento
- ✅ Receber dicas personalizadas com IA
- ✅ Acompanhar o progresso em tempo real
- ✅ Treinar a apresentação final do projeto

---


## 🚀 Como correr o projeto localmente

```bash
# 1. Clonar o repositório
git clone https://github.com/teu-usuario/projectmentor-ai.git

# 2. Entrar na pasta
cd projectmentor-ai

# 3. Instalar dependências do frontend
cd frontend
npm install

# 4. Iniciar o servidor de desenvolvimento
npm run dev
```

> ⚠️ Instruções para o backend e variáveis de ambiente serão adicionadas na Fase 2.
